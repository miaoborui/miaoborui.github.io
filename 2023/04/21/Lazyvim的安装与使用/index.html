nvim && vim

nvim 和 vim 都是十分强大的终端编辑器

他们有十分强大的键位系统

但他们的配置就有点难

config

vim的配置我不喜欢，因为他的配置全在一个文件中

nvim 的是模块化配置,配置直观，但使用的是lua语言配置，所以比较繁杂

vimide

因为配置繁杂，所以vimide出现了，lazyvim也是vimide

lazyvim 安装

前提条件

首先安装nvimsudo pacman -S nvim
安装gitsudo pacman -S git
终端kitty(可选)sudo pacman -S kitty
nerd font (可选但推荐安装) jetbrins Mono Nerd Font

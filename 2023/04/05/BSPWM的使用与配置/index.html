我为什么用窗口管理器——BSPWM

这篇文章中，我会介绍平铺式窗口管理器

平铺窗口管理器有许多：

  1. 有极简主义的dwm
  2. 也有十分好用的i3wm
  3. 有基于wayland的sway
  4. 有动画丝滑的hyprland
  5. 有我正在使用的bspwm

介绍了这么多那什么是平铺式窗口管理器呢？
在Linux的其他桌面中，最重要的就是窗口了，而平铺式窗口管理器则是将窗口按一定规则平铺摆放

好的，那么开始介绍我为什么使用bspwm（大佬请略过）:

  1. 他有较完整的配置方案
  2. 他的配置好读懂
  3. 他没有想dwm一样配置要sudo make clean install

…..（此处省略114514个字）

怎么安装bspwm？

bspwm安装很简单在arch系的Linux中只需要sudo pacman -S bspwm feh picom polybar rofi
其他的后续补充

那么为什么安装这些软件

bspwm：

一个窗口管理器

feh：

你的壁纸

picom：

动画效果（合成器）

polybar：

状态栏配置简单，自由度高

rofi：

程序启动器

开始配置bspwm

bspwm配置分为两部分（也可以是三部分）
一部分是基本配置
一部分是按键配置
（一部分是状态栏）
打开终端在 .config 文件夹中创建bspwm（基本配置）和sxhkd（按键配置）
cd ~/.config
mkdir bspwm
mkdir sxhkd
如果有则省略

基本配置

在 .config/sxhkd 中创建sxhkd文件
打开bspwmrc
应该有一下代码

#!/bin/bash
pgrep -x sxhkd > /dev/null || sxhkd &

$HOME/polybar-collection/launch.sh（请无视这一行谢谢）

bspc monitor -d I II III IV V VI VII VIII IX X

bspc config border_width 3
bspc config window_gap 12

bspc config normal_border_color ‘#282828’
bspc config focused_border_color ‘#81A1C1’
bspc config urgent_border_color “#88C0D8”
bspc config presel_border_color “#8FBCBB”
bspc config presel_feedback_color ‘#81A1C1’

bspc config split_ratio 0.5
bspc config borderless_monocle true
bspc config gapless_monocle true
bspc config focus-follows-pointer true

bspc rule -a Gimp desktop=’^8’ state=floating follow=on
bspc rule -a Chromium desktop=’^2’
bspc rule -a mplayer2 state=floating
bspc rule -a Kupfer.py focus=on
bspc rule -a Screenkey manage=off

如果没有则copy进去
好！配置完了

按键配置

举一个最基本的例子用快捷键打开自己喜欢的终端，我这里的是kitty

打开sxhkd前面就有关于终端的配置我这里修改过

把其中的

kitty

打开rofi
找到类似区域，没有则创建

在

-theme …..

好！配置完了
